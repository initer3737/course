1.Type Data

Type Data Adalah pengelompokan data berdasarkan jenis data tersebut ,tipe data berfungsi agar si kompiler tahu bagaimana data tersebut akan digunakkan

2.String & char

pada setiap bahasa pemrogaman terdapat beberapa type data pada umumnya ada INTEGER (angka) , STRING (huruf) dan CHAR (karakter 1 huruf ) untuk menulis String ketikkan tanda petik 2 seperti ini "ini adalah string" dan huruf maupun angka jika dia sudah diapit tanda petik 2 maka akan dianggap sebagai type data string untuk angka tinggal ketikkan angka saja tanpa tanda petik 2 nb untuk bahasa pemrograman tertentu bisa menggunakkan petik 1 mauppun petik dua untuk mengetikkan type data string maupun char

3.BOOLEAN

type data ini sangat unik yaitu hanya ada 2 keluaran yaitu true dan false type data ini akan sangat berguna sekali untuk membandingkan kedua nilai jika kedua nilai benar maka akan menghasilkan true dan sebaliknya akan menghasilkan false

4.Array

type data array akan sangat berguna sekali jika anda ingin menyimpan data lebih dari 1 analoginya seperti perpustakaan "didalam perpustakaan berderet rapi buku buku tersimpan di dalam rak" anggaplah buku itu sebagai data dan untuk mengakses data anda akan mencari index sesuai data yang akan anda cari nah kurang lebih seperti itulah gambaran mengenai type data array

5.variable

variable adalah tempat menyimpan type data berupa string ,int,boolean,char type data lainnya

6.OPERATOR

operator sangat penting untuk membuat sebuah program baik itu aplikasi web android maupun aplikasi pc ,operator ada beberapa diantaranya tambah (+) kurang (-) bagi (/) kali (*) kurang dari (<) lebih dari (>) kurang dari sama dengan (<=) lebih dari sama dengan (>=) dan operator pembanding (==) sisa bagi atau modulo (%) tidak sama dengan (!=)

#tips : algoritma pemrogaman

sangat penting bagi teman-teman untuk menguasai "logika matematika dasar , operator aritmatika,problem solving,Mengetahui Prinsip Algoritma
